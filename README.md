# Amazon Prime Video Content Analysis

## 🎬 Dashboard Preview
![Amazon Prime Dashboard](amazon_prime_dashboard.png)

## 📌 Project Overview
This project provides a comprehensive look at the content available on Amazon Prime Video. Using Power BI, I transformed raw dataset into an interactive dashboard to visualize how the streaming service has grown over time and what genres dominate the platform.

## 🛠️ Data Engineering & Visuals
* **Data Cleaning:** Used Power Query to split genres (delimited by commas), handle null values in 'Directors' and 'Cast', and standardize date formats.
* **Key Visuals:**
  * **Map Visual:** To show content availability by country.
  * **Donut Charts:** For the split between Movies and TV Shows.
  * **Time Series:** Tracking the growth of content releases by year.
* **Filters (Slicers):** Added interactivity by allowing users to filter by Rating (PG-13, TV-MA, etc.) and Release Year.

## 💡 Top Findings
* **Content Mix:** Identified the ratio of Movies vs. TV Shows (usually Movies dominate on Prime).
* **Growth Trend:** Visualized the massive spike in content added between 2018 and 2021.
* **Top Genres:** Found that Drama and Comedy are consistently the top-performing categories.
